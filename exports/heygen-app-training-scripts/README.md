# BTS App Training Scripts (HeyGen-Ready)

**8 of 8 app training videos**, transcribed, cleaned, BTS-branded, and ready for HeyGen AI avatar narration.

- **Flexy** — 276 words → `scripts/flexy-training.txt`
- **DIYTrax** — 275 words → `scripts/diytrax-training.txt`
- **MetricMover** — 235 words → `scripts/metricmover-training.txt`
- **ScrapeBot** — 226 words → `scripts/scrapebot-training.txt`
- **CropBot** — 258 words → `scripts/cropbot-training.txt`
- **Gifster** — 294 words → `scripts/gifster-training.txt`
- **PixelPress** — 270 words → `scripts/pixelpress-training.txt`
- **BTS Concierge** — 425 words → `scripts/concierge-training.txt`

## What's inside each script file

Each `scripts/<app>-training.txt` file has a metadata header and then the spoken script after the `--- SCRIPT ---` marker. Copy everything below that marker into HeyGen's script field.

## Cleaning applied

- Filler words removed (um, uh, you know, like, okay so, right?)
- Run-on sentences tightened, redundant repetition removed
- Transcription artifacts fixed (MediaMavens, ClickBank, DIYTrax, Flexy, Gifster casing)
- BTS branding swept (any residual TCE / "The Conversion Engine" replaced)
- Original instructional voice and second-person tone preserved
- No new facts, prices, URLs, or steps invented

## Source

These were extracted from the existing Build Test Scale knowledge base (the same 97-script set processed previously) and re-keyed to the seven Apps + Concierge cards shown in the BTS portal at `/advantage` and `/apps`.
